## qecc-sim

**量子誤り訂正シミュレータ**

Surface CodeとColor Codeのシンドローム測定・デコーダをPythonで実装したシミュレータ。

- **技術**: Python, NumPy, Graph theory
- **リンク**: [GitHub](https://github.com)

```python
from qecc_sim import SurfaceCode, MinimumWeightDecoder

code = SurfaceCode(distance=5)
decoder = MinimumWeightDecoder(code)
result = decoder.decode(syndrome=code.random_error(p=0.01))
print(result.logical_error_rate)
```
