# Projects

## Artport

**芸術をもっと身近にするアプリ**

「なんとなく難しそう」という理由で芸術に触れる機会を逃してしまう人を減らしたい、という思いから開発しているアプリ。展覧会をもっと気軽に、もっと楽しく体験できることを目指しています。

- **技術**: React(Javascript), Tailwind CSS, Firebase, Cloudinary, Qwen API
- **リンク**: [GitHub](https://github.com/magarikado0/artport), [アプリURL](https://artport.magarikado.net)
